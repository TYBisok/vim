# Help

在日常的使用过程种难免会要查看文档，查阅某些功能的使用方法；但是有时候也会忘记想要查看的功能的关键字，所以这里主要存放各种功能如何在 Vim 种查询的方法：

## 插入

### 用二合字母插入非常用字符

* `h digraphs-table` - 查找用于 `<C-k>{char1}{char2}` 非常用字符及合成该非常用字符的两个字母

## Ex 命令

* `:h ex-cmd-index` - 查看 Ex命令 列表
